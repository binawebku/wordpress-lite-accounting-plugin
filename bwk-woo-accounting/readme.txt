A lightweight accounting suite for WooCommerce. Provides chart of accounts, journal entries, expenses, and automatic sync of paid orders.

== Description ==
* Dashboard with KPIs.
* Sales & Invoices, Expenses, Chart of Accounts, Journals, Ledgers, Reports.
* Automatic journals on WooCommerce order completion.
* Expense entry with automatic journal posting.

== Installation ==
1. Upload `bwk-woo-accounting` to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' screen.
3. Access accounting via `BWK Accounting` menu.

== Frequently Asked Questions ==
= Does it modify WooCommerce tables? =
No, journals are stored in custom tables.
